# Canva 연동 안내

이 발표 자료는 Canva에서 가져올 수 있도록 편집 가능한 PowerPoint 파일로 만들었습니다.

권장 흐름:
1. Canva를 열거나 요청에 포함된 링크를 엽니다: https://canva.link/c5rqw390fimze98
2. `mini-project-build3-presentation.pptx`를 업로드합니다.
3. Canva의 PowerPoint 가져오기/변환을 완료합니다.
4. 한국어 폰트가 대체되면 Canva 안에서 한글 지원 폰트로 바꿉니다.
5. 이미지, 도형, 텍스트 박스는 가져온 뒤에도 개별 편집 대상으로 확인합니다.

이번 Codex 세션에는 실제 Canva 디자인을 원격 생성/편집하는 호출 가능 도구가 노출되지 않아, Canva에 바로 업로드 가능한 PPTX 산출물로 준비했습니다.
